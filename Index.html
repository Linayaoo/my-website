I-Library management system

The Self-Service School Library Management System is a modern library solution designed to streamline book borrowing, returning, and inventory management. Utilizing RFID technology, GSM modules, and a centralized database, the system enhances efficiency, reduces administrative workload, and improves user experience for National University students and staff.

Smart Plant Watering System

The Automatic Plant Watering System is a smart irrigation solution designed to optimize plant care by automating watering based on soil moisture levels. The system integrates Arduino, soil moisture sensors, a water pump, and a Bluetooth module, allowing users to manage watering through a Windows or mobile application. This project aims to conserve water, prevent overwatering, and ensure plant health even when owners are away.

Bluetooth Remote Car

The Bluetooth Remote-Controlled Car is a compact, wireless vehicle that can be operated via a mobile application. It is designed for automation, robotics learning, and remote navigation experiments. The car utilizes Bluetooth communication to receive commands and control movement, making it an excellent project for IoT, embedded systems, and robotics.

Trash Segregator

The Automated Trash Segregator is a smart waste management system that classifies trash into biodegradable, non-biodegradable, and metallic waste. Using sensors and a microcontroller, the system identifies waste types and sorts them into respective bins, promoting efficient recycling and proper waste disposal.
